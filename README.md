# 🌿 Бот «Дочь Богини» — карта недели

Телеграм-бот с картинками: каждую неделю выдаёт одну из 40 славянских богинь с напутствием и иллюстрацией.

## Как работает

- У каждого пользователя в каждую неделю — **своя богиня** (детерминировано по ID + номеру недели).
- Карта меняется автоматически в понедельник (по ISO-стандарту).
- Пользователь видит картинку богини + подпись с описанием и напутствием.
- Команда `/card` показывает текущую карту.

## Путь A. Быстрый запуск онлайн на Railway (без установки Python)

Это самый простой путь: бот будет работать 24/7 на бесплатном хостинге.

### Шаг 1. Создать бота у @BotFather

1. В Телеграме найдите [@BotFather](https://t.me/BotFather), нажмите **Start**.
2. Отправьте `/newbot`.
3. Введите имя (например, «Дочь Богини»).
4. Введите юзернейм — должен заканчиваться на `bot` (например, `doch_bogini_bot`).
5. **Скопируйте и сохраните токен** — строку вида `1234567890:ABCDEF...`.

### Шаг 2. Создать аккаунт на GitHub (бесплатно)

1. Идите на [github.com](https://github.com) → **Sign up**.
2. Введите почту, пароль, юзернейм. Подтвердите почту.

### Шаг 3. Загрузить код бота на GitHub

1. На github.com нажмите зелёную кнопку **New** (или плюсик вверху справа → **New repository**).
2. Имя репозитория: `doch-bogini-bot` (любое).
3. Поставьте **Public**.
4. Нажмите **Create repository**.
5. На открывшейся странице нажмите **uploading an existing file**.
6. **Перетащите в окно все файлы и папки из этой папки** (`bot.py`, `requirements.txt`, `Procfile`, `.gitignore`, папку `cards/` с картинками).
7. Внизу нажмите **Commit changes**.

⚠️ Убедитесь, что в папке `cards/` лежат все 40 картинок с именами, указанными в `cards/README.txt`. Имена должны быть **латиницей** и с точным регистром.

### Шаг 4. Подключить Railway

1. Идите на [railway.com](https://railway.com) → **Login with GitHub**.
2. Разрешите Railway доступ к вашему GitHub.
3. Нажмите **New Project** → **Deploy from GitHub repo** → выберите ваш `doch-bogini-bot`.
4. Railway начнёт развёртывать бота. Пока он думает — перейдите к настройке токена.

### Шаг 5. Вставить токен

1. В Railway на странице проекта откройте свой сервис (серый блок с именем репо).
2. Вкладка **Variables** → **New Variable**.
3. Имя: `BOT_TOKEN`, значение: ваш токен от BotFather (без кавычек).
4. Нажмите **Add**. Railway автоматически перезапустит бот.

### Шаг 6. Проверить

1. Откройте в Телеграме вашего бота (по юзернейму, который дали в BotFather).
2. Нажмите **Start**.
3. Напишите `/card` — должна прийти картинка богини с напутствием.

✅ **Готово!** Бот работает 24/7. Railway даёт бесплатные 500 часов в месяц — этого с запасом хватит на одного бота.

## Путь B. Запуск на своём компьютере (если хотите сначала протестировать)

Если хотите попробовать локально перед деплоем.

### Windows:
1. Скачайте Python 3.10+ с [python.org](https://python.org) — при установке поставьте галочку «Add Python to PATH».
2. Откройте командную строку (Win+R → `cmd`).
3. Перейдите в папку: `cd C:\путь\к\папке\telegram-bot`
4. Установите зависимости: `pip install -r requirements.txt`
5. Откройте `bot.py` в Блокноте, вставьте токен в строку `BOT_TOKEN = ...`.
6. Запустите: `python bot.py`

### Mac / Linux:
1. Проверьте Python: `python3 --version` (обычно уже есть).
2. `cd путь/к/папке/telegram-bot`
3. `pip3 install -r requirements.txt`
4. Отредактируйте `bot.py`, вставьте токен.
5. `python3 bot.py`

Пока окно открыто — бот работает. Закроете — бот остановится.

## Что делать, если картинки не приходят

Бот отправит только текст, если картинки не найдены. Причины:
- В папке `cards/` нет нужных файлов.
- Имена файлов не совпадают с тем, что указано в коде (чувствительны к регистру!).
- Картинка весит больше 10 МБ — Телеграм не примет. Сожмите через [squoosh.app](https://squoosh.app).

Проверьте в логах Railway (вкладка **Deployments** → **View Logs**): там будет строка `📁 Найдено картинок: N` — должно быть 40.

## Как менять тексты напутствий

Все напутствия — в `bot.py` в массиве `GODDESSES`. Формат:

```python
{
    "name": "Имя",
    "world": "Навь|Явь|Правь",
    "image": "имя_файла.jpg",
    "desc": "Описание богини",
    "message": "Напутствие на неделю",  ← это меняете
},
```

После изменений загрузите файл на GitHub (через веб-интерфейс: откройте файл → карандаш → внесите правки → Commit changes). Railway сам перезапустит бота через минуту.
